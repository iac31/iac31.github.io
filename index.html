Layout

Container

Choose between a fixed width and a full width container to wrap big sections of the website.

Use .container as a fixed width container.

<div class="container"></div>

Container fluid

Use .container-fluid for the entire width of the viewport.

<div class="container-fluid"></div>

Grid system

6 general columns

Each with flex of X

<div class="col-X"></div>

And 3 columns for every breakpoint

<div class="col-BP-X"></div>

BP less than
xs 576px
sm 767px
md 991px
lg 1199px
xl ∞

Parent div

Use .flex class for div parent so that div children to be placed as row

<div class="flex"></div>

For a row section to be placed as columns when viewport is less than the breakpoint, use .BP-column extra class for which breakpoint you want to change the layout. For example, use .sm-column for the layout to change under 767px. The same applies for .BP-flex

<div class="flex sm-column"></div>

Responsive example

For big screens we have 20% sidebar and 80% content, for under md breakpoint (991px) we will have 33% sidebar and 66% content, under sm breakpoint (767px) we will have equal width for the sidebar and content, under xs breakpoint (576px) the items will be displayed as column.

<div class="flex column">
    <nav></nav>
    <div class="flex xs-column">
        <div class="sidebar col-1"></div>
        <div class="content col-4 col-md-2 col-sm-1"></div>
    </div>
</div>
